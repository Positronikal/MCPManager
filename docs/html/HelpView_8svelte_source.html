44 <h3>Welcome to MCP Manager</h3>
45 <p>
46 MCP Manager is a cross-platform application for discovering, managing, and monitoring
47 Model Context Protocol (MCP) servers on your system.
48 </p>
49
50 <h4>Getting Started</h4>
51 <ol>
52 <li>
53 <strong>Discover Servers:</strong> Click the "Refresh" button to scan your system
54 for MCP servers. Servers can be discovered from Claude Desktop configuration,
55 filesystem scans, or running processes.
56 </li>
57 <li>
58 <strong>Start/Stop Servers:</strong> Use the action buttons in the server table
59 to control server lifecycle. Green indicates running, red indicates stopped.
60 </li>
61 <li>
62 <strong>View Logs:</strong> Monitor real-time logs in the bottom panel. Filter
63 by server, severity, or search for specific messages.
64 </li>
65 <li>
66 <strong>Configure Servers:</strong> Click the ‚öôÔ∏è button to edit environment
67 variables, command-line arguments, and behavior settings.
68 </li>
69 </ol>
70

168 <h4>About This Application</h4>
169 <p>
170 MCP Manager is a comprehensive management tool for Model Context Protocol servers.
171 It provides a unified interface for discovering, monitoring, and controlling MCP
172 servers across your development environment.
173 </p>
174
175 <h4>Key Features</h4>
176 <ul>
177 <li>Automatic server discovery from multiple sources</li>
178 <li>Real-time server status monitoring</li>
179 <li>Live log streaming with filtering</li>
180 <li>Configuration management with validation</li>
181 <li>System utilities (netstat, shell, explorer, services)</li>
182 <li>Keyboard shortcuts for efficient workflow</li>
183 <li>Cross-platform support (Windows, macOS, Linux)</li>
184 </ul>
185
186 <h4>Technology Stack</h4>
187 <dl>
188 <dt>Backend</dt>
189 <dd>Go 1.21+ with Wails v2 framework</dd>
190
191 <dt>Frontend</dt>
192 <dd>Svelte 3.x with TypeScript</dd>
193
194 <dt>Architecture</dt>
195 <dd>Event-driven with real-time updates via Wails runtime events</dd>
196 </dl>
